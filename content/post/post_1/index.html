Guía para el uso de emojis en Go Hugo

Uso de emojis

Emoji puede ser habilitado en un proyecto Hugo de distintias formas.

La función emojify puede ser llamada directamente en templates o Inline Shortcodes.

Para habilitar emoji globalmente, cambia ´enableEmoji´ a ´true´ en la configuración de tu sitio y entonces podrás escribir codigos rápidos de emoji directamente en el contenido de tus archivos; por ejemplo:

🙈 :see_no_evil: 🙉 :hear_no_evil: 🙊 :speak_no_evil:


EL EMoji cheat sheet es una referencia muy útil para codigos rápidos de emoji. Emoji cheat sheet

N.B. Los pasos anteriormente mencionados habilitan los catactéres de emoji y sus secuencias en Hugo, sin embargo el renderizado de estos glyphs dependerá del navegador y la plataforma. Para stilizar el emoji puedes usar una fuente externa de terceros o un font stack; por ejemplo:

.emoji {
  font-family: Apple Color Emoji, Segoe UI Emoji, NotoColorEmoji, Segoe UI Symbol, Android Emoji, EmojiSymbols;
}
